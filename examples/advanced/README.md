Advanced Examples
---

The Advanced Examples include scripts that demonstrate further things that can be done using ClearML. 


* [Execute Remotely](execute_remotely_example.py) - script that demonstrates how to use the `task.execute_remotely` method,
  so part of the code can run locally and the rest can be executed remotely.
* [Multiple Tasks](multiple_tasks_single_process.py) - demonstrates how to instantiate multiple tasks in a single script.
