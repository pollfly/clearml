Advanced Examples
---

edThe Advanced Examples include scripts that demonstrate additional tools available in ClearML:
* [Execute Remotely](execute_remotely_example.py) - Using the `task.execute_remotely` method,
  so part of the code can run locally, and the rest can be executed remotely.
* [Multiple Tasks](multiple_tasks_single_process.py) - Instantiating multiple tasks in a single script.
