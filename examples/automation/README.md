Automation Examples
---

The Automation Examples demonstrate methods to automate processes when using **ClearML**.

The examples include: 
* [Manual Random Parameter Search](manual_random_param_search_example.py) - Executing an experiment multiple 
  times, each time with different sets of random hyperparameters.
* [Task Piping](task_piping_example.py) - Creating an instance of a Task from a template Task ([Toy Base Task](toy_base_task.py)), 
  customizing that instance, and enqueuing the customized instance to execute.
